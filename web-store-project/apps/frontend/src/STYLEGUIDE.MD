# Application Style Guide

This document defines the visual design standards for the application to ensure a consistent and unified appearance.

---

## Colour Palette

The following colours should be used throughout the application (3-5):

| Name | Purpose | Hex Value (if applicable) |
1. Primarily Light Gray | Background Light Mode | #c4cdcf
2. Blackish Gray | Alternative Background Color | #212227
3. Dark Turquoise | Dark Accents for Border | #31555b
4. Light Navy Blue | Light Accent for Photo/Logo | #aab9cf


## Typography

The following fonts and sizes are used throughout the application:

### Font Family:
San Francisco Pro; Segou UI
Arial, sans-serif

### Font Sizes (3 minimum)
Header - 1.75rem
Body - 1rem
Footer - 2rem
